# Physical AI & Humanoid Robotics Course - Homepage

## Overview
This Docusaurus-based homepage has been completely redesigned to showcase the Physical AI & Humanoid Robotics Course. The design is modern, futuristic, and specifically tailored to the field of Physical AI.

- Futuristic gradient design with cyberpunk-inspired colors
- Animated floating robot visualization
- Dual call-to-action buttons for learning and reading

### 2. About Section
- Real-world applications of Physical AI
- Advanced robotics concepts
- Industry-ready skills development

### 3. Value Proposition Section
- Hands-on learning approach

- Cutting-edge technology focus

### 4. Comprehensive Curriculum
- Detailed chapter breakdowns
- Interactive curriculum grid with hover effects
- Sequential numbering for easy navigation

### 5. Technology Stack
- AI & Machine Learning tools

### 6. Community Section

## File Structure
- `src/pages/index.tsx` - Main homepage component with all sections
- `src/pages/index.module.css` - Modern styling with animations
- `docusaurus.config.ts` - Updated site configuration

## Navigation
The homepage connects to the existing documentation structure:
- `/docs/intro` - Course introduction
- `/docs/01-introduction-to-physical-ai/intro-to-physical-ai` - Physical AI book
- Other curriculum sections map to corresponding documentation

## Customization
To modify the content:
1. Edit the text in `src/pages/index.tsx` within each section component
2. Adjust colors in `src/pages/index.module.css`
3. Update site metadata in `docusaurus.config.ts`

## Development
To run the site locally:
```bash
cd frontend Physical AI BOOK
npm run start
```
